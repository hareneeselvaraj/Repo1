🔷 4. Functional Augmentation (1.5.1)
________________________________________
4.1 New Capability: Process Stability Monitoring
________________________________________
Description
Evaluate whether a process is:
•	Stable (in control)
•	Unstable (special cause variation)
________________________________________
Inputs
•	Measure values (time-series)
•	UCL / LCL (from 1.3)
•	Mean (calculated or defined)
________________________________________
Capabilities
•	Plot control chart
•	Detect rule violations
•	Highlight instability
•	Provide insight alongside KPI
________________________________________
________________________________________
4.2 SPC Rule Evaluation
________________________________________
Phase 1 Rules (Minimum)
•	Point outside UCL/LCL
•	7 consecutive points trending up/down
•	7 points above/below mean
________________________________________
Output
•	Stability Status:
o	Stable
o	Unstable
________________________________________
________________________________________
4.3 Relationship with KPI
________________________________________
KPI Status = Threshold-based (Target)

SPC Status = Variation-based (Control Limits)
________________________________________
Key Insight
•	KPI can be GREEN but process UNSTABLE
•	KPI can be RED but process STABLE
